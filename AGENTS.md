# HayekSwarm

Decentralized multi-agent intelligence through Hayekian market economics.

## Architecture

- **HayekMAS** — Core economic engine (auctions, payments, evolution)
- **Swarm** — 10-D council, cost router, consensus, Docker sandbox
- **Skills** — 24 production skills
- **Frontend** — Next.js dashboard + Vue 3 UI

## Quick Start

```bash
pip install -e .
python main.py
```

## Key Concepts

- **Auctions**: Agents bid for the right to act. Highest bidder wins.
- **Bucket-Brigade**: Winners pay previous actors. Value flows backward.
- **Economic Selection**: Wealthy agents mutate. Bankrupt agents die.
- **No Central Controller**: Coordination emerges from market incentives.
